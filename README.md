# Product-Details

##Start Up - Dev

npm install
npm run seed
npm dev:build
npm dev:start
currently on: localhost:3009


| Endpoints      | Type    |  Ops                       |
|----------------|---------|----------------------------|
|'/details/:id'  |  GET    |  Gets data for an item     |
|'/details/'     |  POST   |  Adds data for a new item  |
|'details/:id'   |  PUT    |  Updates an existing item  |
|'/details/:id'  |  DELETE |  Deletes an exiting item   |
